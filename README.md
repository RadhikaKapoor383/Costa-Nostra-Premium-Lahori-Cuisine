# Costa-Nostra-Premium-Lahori-Cuisine
Costa Nostra is a responsive website for a restaurant, offering table reservations and food delivery services. Built entirely with HTML, CSS, and JavaScript, it provides a smooth, interactive experience on desktop, tablet, and mobile devices.

Features

Fully responsive design for all screen sizes

Interactive table reservation system

Online food delivery functionality

Clean and modern UI/UX reflecting authentic Lahori cuisine

Dynamic buttons and menu interactions using JavaScript

This project demonstrates front-end development skills with HTML, CSS, and vanilla JavaScript, focusing on responsive layout and user-friendly interactions.
